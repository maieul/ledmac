\begin{abstract}
  This file provides example of use of \verb+sameparallelpagenumber+ option of \emph{eledpar}.
 Before \verb+\Pages+, the pages are normally numbered.
 When parallel pages are typeset, the number is the same on left and right side.
 After that, page number continues in the normal way.
 
\end{abstract}
